Detailed Description

This node clusters laser scan data points and fits each cluster to a circular landmark.

PARAMETERS: distance_threshold: (double) Threshold to determine if a point is in a cluster radius_threshold: (double) Threshold to determine if a cirlce fit is valid for an obstacle frame_id: (string) The frame the Laser scan data is relative to PUBLISHES: /landmark_data: (nuslam/TurtleMap) a list of centers and radii for cylindrical landmarks SUBSCRIBES: /scan: (sensor_msgs/LaserScan) the raw laser data from the turtlebot SERIVCES:
