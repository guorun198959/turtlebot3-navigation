Detailed Description

This node interfaces between a main computer and the raspberri pi on the turtlebot.

PARAMETERS: left_wheel_joint (std::string): the name of the left wheel joint right_wheel_joint (std::string): the name of the right wheel joint wheel_base (double): the distance between the two wheels of the diff drive robot wheel_radius (double): the radius of the wheels frequency (double): the frequency to publish joint states at tvel_lim (double): the maximum translational velocity avel_lim (double): the maximum angular velocity motor_lim (double): the maximum rotation velocity of the motor encoder_ticks_per_rev (int): the number of encoder pulses per revolution of the wheel motor_power (int): the max command value to send the motor PUBLISHES: /wheel_cmd (nuturtlebot/WheelCommands): a command to control the motors on the turtlebot /joint_states (sensor_msgs/JointState): the wheel position and velocities of the turtlebot SUBSCRIBES: /cmd_vel (geometry_msgs/Twist): the twist command from the turtlesim package /sensor_data (nuturtlebot/SensorData): retrives sensor info from the turtlebot
