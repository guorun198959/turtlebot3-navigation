Detailed Description

This node interfaces between a main computer and the raspberri pi on the turtlebot.

PARAMETERS: /rotation/frac_val: proportion of max speed to use /avel_lim: max robot angular velocity PUBLISHES: /cmd_vel (geometry_msgs/Twist): the twist command SERIVCES: /start (std_srvs/SetBool): Starts the movement of the robot given an initial direction. 0 is CCW/Forwards, 1 is CW/Backwards
